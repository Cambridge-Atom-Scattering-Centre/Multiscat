public tred2(nm, n, a, d, e, z)

This subroutine is a translation of the algol procedure tred2, num. math. 11, 181-195(1968) by martin, reinsch, and wilkinson. handbook for auto. comp., vol.ii-linear algebra, 212-226(1971).

this subroutine reduces a real symmetric matrix to a symmetric tridiagonal matrix using and accumulating orthogonal similarity transformations.

public tql2(nm, n, d, e, z, ierr)

This subroutine is a translation of the algol procedure tql2, num. math. 11, 293-306(1968) by bowdler, martin, reinsch, and wilkinson. handbook for auto. comp., vol.ii-linear algebra, 227-240(1971).

this subroutine finds the eigenvalues and eigenvectors of a symmetric tridiagonal matrix by the ql method. the eigenvectors of a full symmetric matrix can also be found if tred2 has been used to reduce this full matrix to tridiagonal form.
