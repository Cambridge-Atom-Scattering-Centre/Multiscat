Energies/Thetas/Phis

UMLAttribute
FortranMultiscat :: DependancyTree :: Multiscat.conf :: Energies/Thetas/Phis

Description

The energy, theta (incoming beam angle) and phi (azimuthal angle) values at which the scattering will be evaluated.

These are defined over 2 lines each. This can be done in 2 ways: (1)

8,0,0
_,

If the first line is given as (initial value, step delta, number of steps) this range of values will be tested and the second line will be redundant. In this case, all combinations of the energy, theta and phi values will be evaluated.

OR (2)

_,0,n (V1,V2,V3...Vn)

If the forst row contains any number, followed by a 0 followed by a number, the values given on the 2nd line will be taken. In this case, all combinations of energy, theta and phi are not taken; the first energy specified will be evalueated with the first theta and the first phi only, the second energy with the second theta and the second phi only, and so on.

NOTE: The two imput types CANNOT be mixed. Doing so will cause the code to hang
