public loadfixedpot(nzfixed, nfc, vfcfixed, fourierfile)

Loads the potential files in (fourierfile). discards the first 5 lines, then reads the resto of the file into the (vfcfixed).

The fourier components are read one at a time, from the bottom of the potential to the top. This means the kx, ky component is read for each z slice before moving on to the next component.

See pot*.in and the fortran code for more information.
public potent(stepzmin, stepzmax, nzfixed, vfcfixed, nfc, vfc, m, z)

Linearly interpolates the potential data.

This allows the fourier components to be evaluated at the exact points required by the Lobatto quadrature, despite the potential file being discrete and not evaluated at the correct points.

Returns the potential matrix evaluated at the Lobatto roots (vfc)
