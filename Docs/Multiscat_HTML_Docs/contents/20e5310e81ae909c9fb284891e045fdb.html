FortranMultiscat :: DependancyTree :: multiscat.inc

Description

This sets hard limits on Multiscat parameters. The acessible parameters are listed here, in order, as attributes. These values are included in all the fortran scripts at compile time.

Attributes

Visibility Name Type Description
public NZFIXED_MAX none

The max number of z points in fixed pot.
public NVFCFIXED_MAX none

The max number of fourier components (fixed, from file)
public nmax none

Maximum number of diffraction channels
public mmax none

Maximum number of z grid points
public nfcx none

max (?) potential fourier components
