FortranMultiscat :: DependancyTree :: scatCond.in

Description

This file holds a list of conditions at which the scattering is to be evaluated. These are stored as a list of triplets, one triplet of numbers per line. The triplets represent the Energy, theta, phi for each set of scattering conditions.

The first line of the file is for comments only.
